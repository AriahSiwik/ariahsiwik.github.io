Ariah's Audio Library

An easy to use and lightweight upgrade to GameMaker's audio system.


Note: GameMaker 2022.8 recently came out which, for the first time in a long time, changed some details of the audio system. I haven't had the time to fully look into this, so if you are running GameMaker 2022.8 or newer, use at your own risk. I will update this library as soon as I have time.
